## Short project description

This is an open-source, machine-checkable certificate package for a lower bound
on the mixed van der Waerden number `w(2;3,39)`. It publishes the coloring
certificate, an independent Python verifier, documentation of the claim, and
GitHub Actions verification so that others can reproduce the check.

## Why Codex would help

Maintaining the project involves improving verification scripts, adding clearer
reproducibility documentation, reviewing issue reports or pull requests about
certificate validation, and keeping the public mathematical record easy to
audit. Codex would help automate test maintenance, documentation updates, and
review of small contributions while preserving the strict claim discipline of
the certificate package.
